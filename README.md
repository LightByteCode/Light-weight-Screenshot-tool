# Lightweight Screenshot tool

使用Python写的学习型和实用性截图软件，正在更新

计划功能：

1.✅全屏截图

2.✅矩形截图

3.✅录制GIF

4.✅录制视频（❌录制声音）

5.❌指定窗口录制（待考虑）

6.✅快捷键

7.✅图形化界面设置

8.❌加水印

更新日志如下

## 0.2

## 1.1 beta

## 1.3 beta
